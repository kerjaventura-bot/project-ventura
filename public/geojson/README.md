# GeoJSON Folder

Tempatkan file GeoJSON Anda di folder ini.

## Cara Penamaan (Rekomendasi)
Agar dapat ditampilkan secara otomatis per jalur/proyek, Anda bisa menamai file GeoJSON sesuai dengan ID Proyek yang aktif, misalnya:
- `proj-1.geojson`
- `proj-2.geojson`

Atau menggunakan nama yang mudah diidentifikasi lainnya. 

File yang diletakkan di sini akan disajikan secara statis oleh Vite dan dapat diakses melalui URL `/geojson/nama_file.geojson` pada aplikasi.
